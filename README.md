 Descripción
Sistema web en Java EE para gestión automatizada de reservaciones de **Nicole's Decoraciones**. Permite registrar clientes, gestionar fechas y prevenir conflictos de agenda.

Problema Identificado
Gestión manual en libreta física causa:
- Conflictos por doble reservación
- Pérdida de información
- Tiempo excesivo en consultas
- Dificultad de acceso remoto

 Solución Propuesta
Sistema web con:
- Formulario digital de registro
- Validación automática de fechas
- Panel administrativo completo
- Base de datos MySQL
- Interfaz web accesible

Arquitectura
- **Frontend:** HTML5, CSS3, Bootstrap 5, JSP
- **Backend:** Java EE 7, Servlets
- **Base de Datos:** MySQL 8.0
- **Servidor:** GlassFish 5+

 Instalación Rápida
1. Clonar repositorio
2. Configurar MySQL con XAMPP
3. Importar en NetBeans
4. Ejecutar en GlassFish
