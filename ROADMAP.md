## Cómo Contribuir al Roadmap

### Sugerir nuevas características:
1. Crear Issue en GitHub con etiqueta `enhancement`
2. Describir el beneficio para el negocio
3. Proponer prioridad (Alta/Media/Baja)

### Reportar problemas:
1. Crear Issue con etiqueta `bug`
2. Incluir pasos para reproducir
3. Adjuntar screenshots si es posible
